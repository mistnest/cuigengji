# 催更姬

**催更姬** 是一个面向网文作者的开源 AI 写作编辑器。

它不是简单的”聊天框套壳”，而是围绕长篇网文创作流程设计：章节、人物、世界观、剧情记忆、上下文编排、模型调用，都服务于一个目标——**让 AI 更稳定地理解你的故事，并持续辅助你写下去。**

![催更姬截图](public/screenshot.png)

## 为什么做催更姬？

现在的大模型已经很会写，但直接用网页版聊天工具写长篇网文，往往会遇到几个问题：

* 写到后面容易忘记前文设定；
* 人物关系、伏笔、世界观会逐渐混乱；
* 每次对话都要重新解释一遍背景；
* 上下文被无关提示词、系统规则、历史聊天污染；
* 商业 AI 写作软件封闭、不透明、价格高，模型也不一定能自由选择。

**催更姬希望解决这些问题。**

它更像是一个为网文作者准备的 AI 创作工作台，而不是普通聊天窗口。

## 核心卖点

### 1. 面向长篇创作的记忆系统

催更姬会对过往章节、剧情信息、人物状态和世界观设定进行整理与压缩，而不是简单把历史正文全部塞进上下文。

相比直接使用网页版 AI，催更姬能更好地维护长期记忆，让模型在创作后续章节时更容易记住：

* 角色经历过什么；
* 剧情发展到哪里；
* 哪些伏笔已经埋下；
* 哪些设定不能冲突；
* 当前章节应该承接哪些前文信息。

这让 AI 更适合参与长篇网文创作，而不是只会写单章短文本。

### 2. 符合作家习惯的网文编辑器形式

催更姬不是纯聊天工具，而是以“网文编辑器”的形式组织创作流程。

作者可以围绕章节、大纲、人物卡、世界书、剧情摘要等内容进行创作，而不是在一长串聊天记录里来回翻找信息。

它更贴近网文作者的真实写作习惯：

* 先有世界观和人物；
* 再有大纲和章节目标；
* 然后逐章推进正文；
* 写作过程中不断更新记忆、设定和剧情状态。

AI 不再只是一个聊天对象，而是嵌入写作流程里的创作助手。

### 3. Agent 式上下文编排，提高缓存命中率

催更姬采用类 Agent 的上下文编排方式，会尽量把稳定内容和变化内容分离。

例如：

* 世界观、人物卡、写作规则等稳定信息，可以长期保持；
* 当前章节要求、近期剧情、作者临时指令等变化信息，则动态更新；
* 不必要的历史聊天、无关提示词和污染上下文的内容，会尽量避免进入核心上下文。

相比酒馆类工具中更容易混杂历史对话、预设、角色卡和临时指令的方式，催更姬能够更有效地提升缓存命中率，降低重复上下文带来的成本。

同时，它也能绕开部分编程 Agent 中不适合写作场景的 rule、系统提示和工具提示污染，让模型更专注于网文创作本身。

### 4. 开源免费，自由导入模型

催更姬是开源免费软件。

相比市面上许多封闭的 AI 写作软件，催更姬不会把作者锁死在某个固定模型或固定平台里。

你可以根据自己的需求接入不同模型：

* 更便宜的模型；
* 更长上下文的模型；
* 更适合中文网文的模型；
* 本地模型或第三方 API；
* 未来更强的新模型。

作者可以自由选择自己的创作工具链，而不是被平台套餐、内置模型和封闭功能限制。

## 催更姬适合谁？

催更姬适合希望用 AI 辅助长篇创作的作者，尤其适合：

* 网文作者；
* 轻小说作者；
* 剧情策划；
* 世界观设定爱好者；
* 想用 AI 管理长篇故事记忆的人；
* 不想被商业 AI 写作软件绑定的创作者。

## 项目目标

催更姬的目标不是取代作者，而是帮助作者更稳定、更高效地完成长篇创作。

作者仍然是导演、编剧和最终决策者。

AI 负责提供灵感、整理记忆、辅助续写、检查冲突、补全细节，而催更姬负责把这些能力组织成一个更适合网文创作的工作流。

**让 AI 不只是会写一章，而是能陪你把故事写下去。**
