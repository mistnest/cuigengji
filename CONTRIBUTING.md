# 贡献说明

感谢你愿意帮助改进催更姬。

## 许可约定

提交 issue、pull request、补丁、代码、文档、测试用例或其他贡献，即表示你确认：

1. 你有权提交这些内容；
2. 这些贡献可以并入催更姬项目；
3. 项目维护者可以在本项目及其后续版本中使用、修改、分发、再许可和商业授权这些贡献；
4. 这些贡献默认适用仓库根目录的 `LICENSE`。

如果你不同意这些条款，请先不要提交贡献，可以先开 issue 讨论。

## 基本要求

- 不提交真实 API Key、个人项目数据、调试日志或未脱敏截图。
- 不提交 `data/`、`debug/`、`test-results/`、`playwright-report/` 等运行产物。
- 修改功能时尽量补充对应测试，至少说明手动验证方式。
- 讨论产品方向时，请优先说明用户场景和预期行为。
