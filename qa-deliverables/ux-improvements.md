## 1. ✅ 状态栏反馈太弱 — 已修复

**修复方案：** 保留底部状态栏文字的同时，新增 toast 弹出通知（右上角浮层，带滑入/淡出动画）。
- success=绿底 #2d8c4a、error=红底 #d14343、warn=橙底 #c7821a、info=蓝底 #2b6cb0、loading=紫底 #5b3cc4
- 自动 6 秒后消失（loading 12 秒），不影响原有状态栏行为

**修改文件：** `public/js/app.js` `setStatus()`、`public/css/editor.css` toast 样式、`public/index.html` toast 容器

---

## 2. ✅ 缺少新用户引导 — 已修复

**修复方案：** AI 面板 hero 区域下方新增 "快速上手" 3 步引导卡片：
```
1. 选择模型服务商  → 在 #ai-provider 变更时自动 ✓
2. 输入 API Key     → 在 #ai-api-key 输入时自动 ✓
3. 点击「续写正文」 → 测试连接成功后自动 ✓
```
每步完成时圆圈变绿 + ✓ 图标出现，三步全部完成后引导卡片自动隐藏。

**修改文件：** `public/index.html` 引导 HTML、`public/css/editor.css` 引导样式、`public/js/app.js` `updateOnboardingSteps()`

## 4. ✅ 设置按钮功能确认 — 已确认可用

**确认结果：** `showSettingsDialog()` 已完整实现，3 个标签页功能齐全：
- **编辑器** — 主题切换（暖色/清爽/深色）、字体（宋体/黑体/等宽）、字号（14-24px）、行距、字数/Token 显示开关
- **AI 默认** — 默认世界书参考模式、默认角色参考模式
- **数据与调试** — 自动保存/聊天记录/调试按钮开关、Prompt 预览、清理缓存

**结论：** 无需修改，保持现状。
